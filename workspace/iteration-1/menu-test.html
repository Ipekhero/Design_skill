Menu — 4 variants × 2 sizes × 5 states

Menu items reuse the standard row surface family (surface/hover, surface/pressed, surface/selected). The container is always a white floating surface with shadow-lg (Elevation/2, directional). The first cell in each row uses real CSS :hover / :active — hover it to verify the natural state matches the forced state next to it. Selected uses brand-blue softer (#EFF5FA), NOT Figma's teal (#E7F7F9) per the web colour rule.
