Dropdown — all 9 states × 2 layouts × 2 sizes

Three layout combinations across three panels: Vertical Large, Vertical Small, and Horizontal Small. Each shows all 9 states. The selected (menu-open) state has extra cell space so the floating menu fits. Menu's selected row uses color/surface/brand/softer (#EFF5FA — brand blue) instead of Figma's teal list item/selected, per the web rule. Trailing arrows use the just-added arrow_triangle_down / arrow_triangle_up assets, byte-for-byte from assets/icons/.
