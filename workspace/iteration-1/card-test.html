Card — all variants × all states

Three variants stacked top to bottom (Rectangle, Square, Expandable). Each section shows all states side-by-side: default, hover, pressed, and (for Expandable only) expanded. Hover and pressed are force-rendered via .is-hover / .is-pressed; the first cell uses real :hover / :active so you can verify natural CSS matches the forced state. The Square card's icon-wrapper uses brand blue (color/surface/brand/softer + color/icon/brand), not Figma's teal — per the web brand rule.
