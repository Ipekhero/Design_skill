Toggle — 4 states × 2 switchOn values

Binary on/off switch for changes that take effect immediately. Two orthogonal axes: state (default / hover / pressed / disabled) and switchOn (true / false). The first cell in each row uses real CSS :hover / :active — hover it to verify the natural state matches the forced one next to it. ON track uses brand blue (#1047A1), same as checkbox / radio. The 8×8 OFF knob is intentionally brand-blue — it previews the ON colour.
