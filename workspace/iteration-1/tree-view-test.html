Tree View — 3 states + indentation + realistic trees

Hierarchical rows for nested data. Two row kinds: expandable parents (chevron, toggles open/closed on click) and non-expandable leaves (no chevron, reserve the 24px slot so titles align). Selected is text-only — bold + info-blue, no background tint. Each level adds 24px of left padding. Hover any row to see surface/hover #F3F5F8 tint.
