Radio — 4 states × 2 selected values

The Radio Button has two orthogonal axes: state (default / hover / pressed / disabled) and selected (yes / no). The first cell in each row uses real CSS :hover / :active — hover it to verify the natural state matches the forced state next to it. The ring is always 20×20, the inner dot is 10×10. The selected fill uses brand blue (#1047A1), NOT Figma's dark #16191C — same call as checkbox, per the web colour rule. On UNSELECTED radios, the border stays constant across states — but the inner fill tints to surface/hover (#F3F5F8) on hover and surface/pressed (#E1E7ED) on press.
