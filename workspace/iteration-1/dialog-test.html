Dialog — single variant · header variations · content compositions

The Dialog has only one variant (default) and one rendered state (open). Width fixed at 676px, height content-driven. Each example below sits on a faux scrim (background/overlay dark) to mimic the real modal context. The shadow is the directional 2-layer Figma Elevation/3 — explicit exception to design.md's "shadows are pure-vertical" rule. All SVG icons are inlined byte-for-byte from assets/icons/.
